# Job Skill Gap Analysis & Upskilling Recommender (Indian Job Market)

This mini project analyzes a user's skills and compares them with
Indian job market requirements to identify skill gaps and recommend
courses, certifications, and projects.

## Tech Stack
- Python
- NLP (spaCy, Sentence-BERT)
- Machine Learning
- FastAPI

## Target Users
- Indian Freshers
- Working Professionals


## To run the code 
- python -m uvicorn backend.app.main:app --reload
